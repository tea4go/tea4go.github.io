# 更新日志

## v5.5.7
1. 内置 10套 主题，左下角主题切换可右键切换主题
2. 新增工具安装功能
3. 新增显示配置，支持侧边栏模块启用设置
4. 翻译以及系统设置字体选项按照字体效果渲染
5. 快速命令支持拖拽到临时命令行，临时命令行支持伸缩
6. 临时命令行新增发送按钮，快速命令栏增加终止按钮
7. 收藏命令重构，全局共享，支持一键清空快速命令、系统类别显示和复制新增。
8. 终端、笔记页签支持使用 AIChat 抽屉功能
9. AIChat 消息支持复制到笔记
10. AIChat 支持系统提示词和提示语优化器功能
11. 翻译原文和译文支持复制，原文支持清空
12. 状态栏新增快捷笔记抽屉弹框
13. 消息导航支持搜索
14. 支持系统主题和终端主题自定义
15. AIChat 新增自动清除上下文配置
16. MCP 列表添加反选和批量关闭服务功能
17. AIChat 支持本地文件进行对话
18. 云上笔记支持全量导出到电脑
19. 快捷键支持全部重置为默认

## v5.3.7

2. 收藏命令支持分组展示、支持「钉住」置顶关键指令

4. 终端分配支持快捷键切换
5. 修复老版本 Vim 首次粘贴丢失字符问题
6. 修复 Windows 本地文档复制失效
7. 修复 AIChat 粘贴文本被覆盖
8. 界面优化
